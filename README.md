# Cmake-Example
A Minimal C++ project structure build with cmake and Catch lib for tests.

# Building

Just type on terminal
```
mkdir build && cd build
cmake ..
make
```

## Building Tests
Just type on terminal
```
mkdir build && cd build
cmake ..
make tests
```

# Structure

The project structure is minimalist, for you start to make something amazing.

.
├── cmake_example  
│   ├── resource  
│   ├── src  
│   └── test  

## Catch
